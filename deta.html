الأسبوع 1: أساسيات JavaScript الحديثة

  • وحدات JavaScript الحديثة

  • مقدمة إلى React وJSX

  • نظام توجيه React
  • إدارة الحالة باستخدام Redux
  • Hooks المتقدمة

  • مقدمة في Node.js وNPM

  • المصادقة والأمان
  • قواعد البيانات مع MongoDB وMongoose
  • اختبار واجهات برمجة التطبيقات
الأسبوع 6-7: التكامل ونشر التطبيقات
  • دمج React مع Node.js
  • إدارة الحالة على مستوى التطبيق
  • نشر تطبيقات MERN
  • تحسين الأداء والأمان
  • أفضل الممارسات في تطوير الويب
الأسبوع 8: المشروع النهائي والعرض
  • تطوير تطبيق ويب كامل
  • مراجعة الشفرة وأفضل الممارسات
  • تقديم المشروع والعرض
  • نصائح لمواصلة التعلم والتطوير المهني

  • معرفة أساسية بـ HTML وCSS وJavaScript
  • خبرة أساسية في البرمجة
  • جهاز كمبيوتر مع اتصال إنترنت جيد
  • الرغبة في التعلم والتطوير المستمر

ماذا ستتعلم في هذه الدورة؟

  • بناء تطبيقات ويب تفاعلية باستخدام React
  • إنشاء واجهات برمجة تطبيقات RESTful باستخدام Node.js وExpress
  • التعامل مع قواعد البيانات باستخدام MongoDB
  • تنفيذ أنظمة المصادقة والأمان
